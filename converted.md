# PHYSICS PROJECT
Ultrasonic Distance Measurement
Using Arduino.

<!-- Slide number: 2 -->
# Circuit connection :

![](ContentPlaceholder4.jpg)
VCC to 5v
GND to GND
TRIG to D9
ECHO to D10

<!-- Slide number: 3 -->
# Characteristics :
Measures distance using HC-SR04 ultrasonic sensor.
Uses sound waves (40kHz) to detect objects.
Simple, low-cost, highly used in robotics and automation.

<!-- Slide number: 4 -->
# Components Required:
Arduino Uno
HC-SR04 Ultrasonic senor
Breadboard
Jumper wires
USB Cable
Software to run the code

<!-- Slide number: 5 -->
# Working Principle :
Trigger pin send ultrasonic pulse.
Sound hits object and returns.
Echo pin receives signal.
Arduino calculates Time.
Distance=(Time x 0.034)/2
